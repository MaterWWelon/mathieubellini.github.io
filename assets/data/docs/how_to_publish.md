## Publier Power BI / Tableau

- **Power BI** (Public) : Publier → Obtenir le lien → coller dans `data/projects.json` (champ `link_demo`).
- **Tableau Public** : Publier → Copier le lien partagé → coller idem.
- Pour garder privé: place plutôt un lien `link_repo` vers une démo gif, ou un readme.

## Tips

- Garde les images ~1200×675 (paysage), poids < 200 KB.
- Utilise des verbes d’impact dans `highlights` / `impact`.
